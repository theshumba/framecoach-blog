# FrameCoach Blog

The official blog for **[FrameCoach](https://framecoach.io)** — a free, real-time camera coaching app for indie filmmakers and content creators. FrameCoach helps you learn professional camera techniques while you shoot, with on-screen guidance for framing, exposure, and composition.

This blog covers camera settings, filmmaking tips, visual storytelling, and practical guides for creators at every level. It is part of the FrameCoach content ecosystem, supporting filmmakers with free educational resources.

**Live at:** https://theshumba.github.io/framecoach-blog/

## What Is FrameCoach?

FrameCoach is a free, real-time camera coaching app for indie filmmakers and content creators. It provides on-screen guidance to help you master professional cinematography techniques — from framing and composition to exposure and camera movement. Whether you are shooting your first short film or levelling up your content creation, FrameCoach is your on-set coaching companion.

## About This Repo

This repository contains 50+ SEO-optimised blog posts covering filmmaking topics such as:

- Camera settings and exposure guides
- Composition and framing techniques
- Filmmaking gear reviews and recommendations
- Visual storytelling and cinematography tips
- Indie filmmaking workflows

The blog is built with Jekyll and deployed automatically to GitHub Pages.

## Adding New Posts

1. Add a markdown file to `_posts/` with format `YYYY-MM-DD-slug.md`
2. Include front matter (title, description, date, categories, tags)
3. Push to main — GitHub Actions auto-deploys

## Scheduled Publishing

Drop posts in `_drafts/` with future dates. GitHub Actions checks daily at 8am UTC and moves due posts to `_posts/`.

## Structure

- `_posts/` — Published blog posts
- `_drafts/` — Scheduled future posts
- `_pages/` — Hub pages (Camera Settings, Learn Filmmaking, Filmmaker Tools)
- `_layouts/` — Page templates with SEO tags
- `assets/` — CSS and images
