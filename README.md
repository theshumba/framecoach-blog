# FrameCoach Blog

The official FrameCoach filmmaker blog — camera settings, filmmaking tips, and visual storytelling.

**Live at:** https://theshumba.github.io/framecoach-blog/

## Adding New Posts

1. Add a markdown file to `_posts/` with format `YYYY-MM-DD-slug.md`
2. Include front matter (title, description, date, categories, tags)
3. Push to main — GitHub Actions auto-deploys

## Scheduled Publishing

Drop posts in `_drafts/` with future dates. GitHub Actions checks daily at 8am UTC and moves due posts to `_posts/`.

## Structure

- `_posts/` — Published blog posts
- `_drafts/` — Scheduled future posts
- `_pages/` — Hub pages (Camera Settings, Learn Filmmaking, Filmmaker Tools)
- `_layouts/` — Page templates with SEO tags
- `assets/` — CSS and images
